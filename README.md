# License Plate Detector

A web application for detecting and recognizing license plates in images.

## Features

- Upload an image and detect license plates
- Display recognized license plate numbers
- Store detected license plates in a database

## Technologies Used

- Frontend: Vue.js, Tailwind CSS
- Backend: Python, Flask
- Database: SQLite
